## 🎨 Дизайн-система

### Стиль: **Dark Luxury**
| Элемент | Описание |
|---------|----------|
| **Фон** | Rich Black (#0A0A0A) с градиентами |
| **Акцент** | Classic Gold (#D4AF37) |
| **Тени** | Мягкие с золотым свечением |
| **Границы** | Тонкие золотые линии |
| **Эффекты** | Glow, gradient text, floating animation |

### Цветовая палитра
| Цвет | Назначение | Hex |
|------|------------|-----|
| Rich Black | Основной фон | `#0A0A0A` |
| Charcoal | Вторичный фон | `#141414` |
| Classic Gold | Акценты, кнопки | `#D4AF37` |
| Bright Gold | Hover эффекты | `#F4D03F` |
| Deep Gold | Тени, границы | `#B8941F` |
| Pure White | Текст | `#FFFFFF` |
| Light Gray | Вторичный текст | `#B8B8B8` |

### Типографика
- **Заголовки:** Cormorant Garamond (премиальный serif)
- **Текст:** Montserrat (современный sans-serif)

### Эффекты
- ✨ Градиентный золотой текст
- ✨ Золотое свечение (glow)
- ✨ Плавающая анимация карточки
- ✨ Золотое конфетти
- ✨ Плавные transitions (150-300ms)

## ✅ UX Guidelines

Проект соответствует лучшим практикам UI/UX:

- [x] `cursor: pointer` на кликабельных элементах
- [x] Hover states с smooth transitions
- [x] Text contrast 4.5:1 minimum (WCAG AA)
- [x] Focus states visible для keyboard nav
- [x] `prefers-reduced-motion` respected
- [x] Responsive: 375px, 768px, 1024px, 1440px
- [x] Семантическая HTML5 разметка
- [x] ARIA labels для accessibility

## ♿ Доступность

### Поддержка
- ✅ Screen readers (ARIA labels)
- ✅ Keyboard navigation (Tab, Enter, Escape)
- ✅ Focus management в модальных окнах
- ✅ Reduced motion preference
- ✅ Высокий контраст текста

### Клавиатурная навигация
| Клавиша | Действие |
|---------|----------|
| `Tab` | Переключение между кнопками |
